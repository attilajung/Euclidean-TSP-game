# 🎮 [Play the TSP Game](https://attilajung.github.io/Euclidean-TSP-game/)

A fun browser-based puzzle where you find the shortest path connecting random dots. Connect all points exactly once in a loop and try to match the computer's optimal solution!

## What's Included

- **Click to connect**: Select points and link them together to form a path
- **Live feedback**: Watch your distance and see which points need work (color-coded)
- **Hints**: Get suggestions when you're stuck
- **Adjustable difficulty**: Slide to add more (or fewer) points
- No dependencies—just open it in your browser

## Tech

Built with vanilla HTML5, CSS3, and JavaScript. Uses a 2-Opt algorithm to find near-optimal solutions.
